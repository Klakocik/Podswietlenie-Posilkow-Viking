W tym folderze należy umieścić następujące pliki obrazów:

1. settings.png - Zrzut ekranu strony z opcjami/ustawieniami wtyczki
   Rozmiar zalecany: 800x600 pikseli

2. example.png - Zrzut ekranu pokazujący przykład działania wtyczki, 
   podświetlone posiłki z wysoką zawartością białka na stronie panel.kuchniavikinga.pl
   Rozmiar zalecany: 800x600 pikseli

3. Ikony wtyczki:
   - icon16.png (16x16 pikseli)
   - icon48.png (48x48 pikseli) 
   - icon128.png (128x128 pikseli)

Dodaj te pliki przed opublikowaniem wtyczki na GitHub lub Chrome Web Store.
